Install Node JS from the exe in the root folder

Install Bower Package Manager by going to command prompt/terminal and typing the following;
npm install -g bower

Install Grunt JS by typing the following command;
npm install -g grunt-cli

Install Bootstrap with the following command;
bower install bootstrap
